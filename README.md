# BuguerHouse
Hamburgueria ficticia criada atráves de IA utilizando métricas de negócio criadas atráves de prompts em GPT-5 e Python utilizando libs como randon. Tratamento e ETL de tabela para camada gold e conexão com Power BI para construção de Dashboards e KPIS.

Material possui:

- Script em Python para gerar tabela
- Tratamento SQL em Databricks para tratamento e ETL
- Dashboard desenvolvido
- Documentação
